# EV04 – Taller Control Condicional en C#

## Descripción
Este proyecto corresponde a la actividad EV04 – Aplicación de estructuras condicionales en C#.

El programa:
- Solicita la edad del usuario y determina si es mayor o menor de edad.
- Solicita un número y determina si es positivo, negativo o cero.

## Ejecución

Para ejecutar el proyecto se utilizó el siguiente comando:

dotnet run
